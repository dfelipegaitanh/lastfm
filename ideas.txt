Comando necesita tener las siguientes opciones
    * user (Working)
    * yearInit (default, 2006)
    * monthInit (default 4, April)
    * yearEnd (default, actual year)
    * monthEnd (default, actual month)
    * limit (Working)
    * minPlays (Working)

Imports
    * Semanales
    * Mensuales
    * Trimensuales
    * Semestrales
    * Anuales
    * Love Songs (done)
    * Tags (Done)

Base de datos
    * LastFM
        * Artist (Done)
            * Events for plays counts history
        * Song (Done)
            * Events for plays counts history
        * Tags
            * Artist (Done)
            *
        * Time Periods
            * Date Init
            * Date Fin
            * Description (Monthly, Weekly, Yearly)
            * User
        * UserTopTrack
            * User
            * TimePeriod
            * Track
            * Count
